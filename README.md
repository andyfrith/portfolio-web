[![Product Name Screen Shot][product-screenshot]](https://github.com/andyfrith/portfolio-web)

Next.js is "The React Framework for the Web" and perfect for building a portfolio website. Shadcn provides excellent components. TailwindCSS makes styling with CSS a breeze.
